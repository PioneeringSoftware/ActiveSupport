41 /**
42  * Walks a JSON object and also checks the given JSON object for circular
43  * references, failing if circular.
44  * @param object The JSON object to walk, where objects have one of three types:
45  * array, dictionary or primitive. Primitives include strings, numbers and
46  * date-times. The root JSON object can be any one of these three, as can array
47  * elements and dictionary objects. Dictionary keys can be strings only.
48  * @result Answers an array of unique objects extracted by walking the JSON @a
49  * object. The array contains all primitives, all arrays and their elements, and
50  * all dictionaries and their values. The array does @em not contain dictionary
51  * keys. You can extract dictionary keys from the result by picking out the
52  * dictionaries. Answers `nil` if the JSON object contains circular references.
53  *
54  * Walks the JSON object, adding each object to the resulting array
55  * one-by-one unless it encounters an object already seen. Checking for circular
56  * references uses the <code>-[NSArray firstObjectCommonWithArray:]</code>
57  * method. If using an array is not efficient, the implementation could improve
58  * by using a set rather than an array. As a disadvantage in that case, the
59  * ordering would become undefined. Dictionaries already make their ordering
60  * undefined.
61  */
62 NSArray *ASAllObjectsInJSONObject(id object);
