29 /**
30  * Converts an RFC 3339 date-time string to a date.
31  *
32  * <a href="http://www.ietf.org/rfc/rfc3339.txt">Request for Comments
33  * 3339</a> describes a standard Internet date-time format as follows.
34  * `yyyy-mm-ddThh:mm:ssZ` where `yyyy-mm-dd` describes the year, month
35  * and day of month using decimal digits; `hh:mm:ss` describes hours, minutes
36  * and seconds. Time is expressed in Zulu time, more commonly known as Greenwich
37  * Mean Time. See Apple's Technical Q&A QA1480 for more details.
38  *
39  * @par Implementation Notes:
40  * The implementation searches for the first successful formatter where the
41  * resulting date reverse-formats to a matching date string. This may result in
42  * a number of formatting and parsing iterations and therefore runs slower than
43  * a one-shot attempt. However, the algorithm effectively guarantees reverse
44  * translation if and when that becomes necessary. To mitigate the iterative
45  * approach, the implementation caches the date formatters.
46  */
47 NSDate *ASDateFromRFC3339String(NSString *string);
